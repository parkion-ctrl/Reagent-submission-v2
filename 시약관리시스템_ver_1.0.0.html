⚙️기술 스택
Python 3.x Django PostgreSQL Jinja2 템플릿 Bootstrap 5 openpyxl psycopg
  • PostgreSQL 다중 스키마 구조 — 부서별 독립 데이터 (dlab, path, nm, haz)
  • Django ORM + 직접 SQL 혼용 (스키마 라우팅 커스텀 레이어)
  • 부서(스키마) 전환 시 미들웨어가 search_path 자동 설정
📄페이지 구성
페이지URL설명
시약 재고/inventory/현재 재고 현황 조회, 엑셀 export
시약 마스터/master/시약 등록·수정·삭제·폐기, 엑셀 일괄 업로드
입고/inbound/개별 입고 처리, 엑셀 일괄 입고
출고/outbound/개별 출고 처리, 엑셀 일괄 출고
입출고 이력 조회/history/이력 조회·필터·엑셀 export, 무상입고/가출고 처리
시약 이력 조회/reagent-history/시약별 라이프사이클 이력, 개봉일·병행 관리, Old/New Lot
청구 내역/billing/월별 무상입고·가출고 집계, 엑셀 export
관리자 허브/admin-panel/사용자·그룹·파트 관리, 이력 수정, 재고 수정

재고 수정/inventory-admin/관리자 전용 현재 재고 직접 수정 (입출고 이력 자동 생성)
🧪시약 관리
  • 시약 마스터 등록 / 수정 / 삭제 / 폐기
  • 엑셀 일괄 업로드 (미리보기 → 확정)
  • 유효기간 자동 만료 처리 (sync_expired_reagents)
  • Old / New Lot 구분 관리
  • 개봉일·병행 사용일 기록
  • 유해화학물질 여부 관리
📦입출고
  • 개별 입고 / 출고 처리
  • 엑셀 일괄 입고 / 출고 (미리보기 → 확정)
  • 가출고(PROV_OUT) — 재고 차감 없이 기록
  • 무상입고(FREE_IN) 처리
  • 입출고 시 현재 재고·필요 수량 자동 재계산
  • 등록자 이름 자동 기록
📋입출고 이력 조회
  • 구분 필터 (입고 / 출고 / 무상입고 / 가출고) — 테이블 헤더
  • 업체 필터 — 테이블 헤더, (공란) 옵션 포함
  • 시약 구분 필터 — 테이블 헤더
  • 파트 / 기간 / 키워드 필터
  • 엑셀 export (현재 필터 조건 유지)
  • 무상입고 badge 청록색 표시
🛠️관리자 기능
  • 입출고 이력 수정·삭제 → 재고 자동 재계산
  • 재고 직접 수정 → 입출고 이력 자동 생성
  • 사용자·그룹·파트 관리
  • superuser 기본 파트 필터 전체 적용
  • 수정·삭제 후 필터 상태 유지
🗄️주요 데이터 구조
테이블주요 컬럼
inventory part, item_code, item_name, lot_no, expiry_date, current_stock, safety_stock, required_qty, reagent_type, equipment, vendor, hazardous, disposed_at, lot_status
transaction_history inventory_id, tx_type (IN/OUT), qty, tx_date, billing_type (FREE_IN/PROV_OUT), remaining_stock, item_code, item_name, lot_no, part, unit, created_by, note
UserProfile user (FK), employee_no, part, department
Part code, name, schema_name
📌기타 사항
  • 시약 이력 조회의 등록자 사번은 표시하지 않음 (이름만 기록)
  • 기존 데이터의 등록자 공란은 fix_created_by.py 스크립트로 일괄 보정 가능 (superuser / 0000000)
  • 재고 수정 페이지의 변경값은 저장 버튼 클릭 → 미리보기 확인 → 예/아니오 후 반영
  • 가출고(PROV_OUT) 처리 이력은 재고 재계산 시 제외됨
  • 유효기간 9999-12-31 은 UI 상 공란으로 표시
